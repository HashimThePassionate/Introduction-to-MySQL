## 4. Boolean datatypes:

- BOOLEAN -- TRUE or FALSE
- BOOL -- 1 or 0.
  ```sql
  CREATE TABLE example_boolean_table (
    boolean_column BOOLEAN,
    bool_column BOOL);
  ```
